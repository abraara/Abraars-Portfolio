What it does and how I made it happen.

The Upstream Oil & Gas Simulation is a complex software application that aims to provide a hands-on approach to learning about the oil and gas industry. The project is designed to teach users how to make informed investment decisions by simulating real-world scenarios that they might encounter in the industry.

As a software developer working on the project, I have been responsible for the front-end design using Javascript, JQuery, and Bootstrap CSS. This involves creating an intuitive user interface that is easy to use and navigate, as well as visually appealing. I have also been responsible for the back-end implementation of users and simulations using C# and the .NET framework. This involves developing the logic and functionality of the simulation, as well as the user management system. I have also worked on migrating the entire repository off a smaller server onto a bigger server that can handle many users at the same time.

The project is being developed by a team of developers and designers, who are working together to ensure that the software meets the client's requirements using the SPRINT system for meetings and updates, Github for merging and pushing, and Jira for ticketing issues/bugs. The client plans to use the software to help African governments gain a better understanding of the oil and gas industry in the West, which will ultimately help them catch up. The software will be used as a teaching tool, providing users with a virtual environment in which they can learn about the industry without the risks associated with real-world investment decisions.

The simulation includes a range of scenarios that users can explore, each of which is designed to teach specific concepts related to the industry. The scenarios are based on real-world data, and users are able to interact with the simulation in a variety of ways. The software also includes a live spreadsheet that allows users to track their progress and identify areas where they need to improve.

What it does and how I made it happen.

Flair Hijab is an e-commerce website that was fully designed and created by me from scratch for a client based in Edmonton, Alberta. The website was designed and developed using the React framework with the Next.js framework, providing a fast and optimized server-side rendering experience. TailwindCSS was used for styling, which allowed for efficient and quick development of the website's layout and design.

The primary focus of the project was to create a fully functional e-commerce website that would enable customers to purchase ethnic clothing, jewelry, and hijabs securely and efficiently. To achieve this, the website integrated the Stripe API as the backend payment system. Stripe is a secure and reliable payment platform that enables businesses to accept payments from customers worldwide, making it a perfect fit for the client's business needs.

The final result was a mobile-friendly, fully functional e-commerce website that met the client's business goals and provided customers with a seamless shopping experience.

What it does and how I made it happen.

Hockey Players is a fully responsive website designed to view information about players in the NHL. The website is built using PHP and MySQL, making it dynamic and interactive, while bootstrap css provides an attractive and user-friendly design. jQuery is used to enhance the website's functionality by providing various features, including filtering and searching.

One of the key features of Hockey Players is the ability for an admin to add, edit, or delete players by using PHP and MySQL. The admin dashboard is secure and requires authentication to ensure only authorized individuals have access to the site's backend.

The website is designed to be easy to use and navigate, providing a seamless experience for visitors. Whether browsing through player profiles or searching for specific information, users will find the website to be both engaging and informative. Overall, Hockey Players is a comprehensive resource for NHL fans looking to stay up-to-date with their favorite players and teams.
